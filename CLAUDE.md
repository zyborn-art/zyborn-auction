# ZYBORN Auction Platform

## Overview
Online auction platform at auction.zyborn.com for art and collectibles.

## Stack
- Framework: Vite + vanilla JS
- Backend: Firebase (Firestore, Auth)
- Hosting: Vercel
- Domain: auction.zyborn.com

## Architecture
- /src/ - Application source code
- /public/ - Static assets
- Firebase Firestore for auction data, bids, users
- Firebase Auth for user/bidder verification

## Key Features
- Bidder verification system
- Real-time bid updates
- Auction listing management
- Seller onboarding

## Deployment
- Auto-deploys from GitHub main branch via Vercel
- Firebase project linked for backend services
